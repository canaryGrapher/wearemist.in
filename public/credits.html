Technologies

MongoDB

MongoDB is a general purpose, document-based, distributed database. We use MongoDB to store all necessary data.

Node.js

Node.js is an open-source, cross-platform, JavaScript runtime environment that executes JavaScript code outside a web browser. The entire backend of the website is written in node.

Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. The amazing look of the website and the fast development was possible because Bootstrap 4.

Ajax

Ajax is a set of web development techniques using many web technologies on the client side to create asynchronous web applications. Ajax was used at locations where it was necessary to fetch and send data without refreshing the page.

Wow.js

Wow.js is a library which helps simplify including animations to elements in the DOM. All the animations that you see on this website was created using this library.

Animate.css

Animate.css is a library of ready-to-use, cross-browser animations for you to use in your projects. Wow.js requires Animate.css to function.

Pug

Pug is a template engine for Node and for the browser. It compiles to HTML and has a simplified syntax, which can make you more productive and your code more readable. The "BIOS" was written in PUG.

HTML and CSS

Hypertext Markup Language is the standard markup language for documents designed to be displayed in a web browser. Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS was used to provide customization that was not possible/easy using bootstrap. HTML is the backbone of frontend.

Javascript

JavaScript is high-level, often just-in-time compiled, and multi-paradigm programming language. The news page and the home page all use Javascript for customization, and the geek page is majorly wriiten javascript, with minimal HTML and CSS.

Sources

Unsplash

Almost all the images were taken from unsplash.com because they are the source of free images, without the need for paying royalities.

Google Fonts

Google Fonts is library of free licensed font famillies. Some custom fonts were imported from fonts.google.com to make it match it with the design of the page..

Wikipedia

Wikipedia is a free online encyclopedia for the internet users. We used wikipedia to find most of the content for the non-geek homepage.
